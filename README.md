**Bahjat Shell** - A Simple Unix Shell Implementation

## Installation
To install and use **Bahjat Shell**:

1. Clone the project:
    ```bash
    git clone https://github.com/yourusername/bahjat-shell.git
    ```

2. Navigate to the project directory:
    ```bash
    cd Bahjat_Shell
    ```

3. Build the project using `make`:
    ```bash
    make
    ```

4. Run the shell:
    ```bash
    ./Bahjat_Shell
    ```
